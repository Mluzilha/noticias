# 📰 Pulse News

Um portal de notícias minimalista e responsivo desenvolvido com HTML, CSS e JavaScript. O projeto consome notícias em tempo real utilizando uma API de notícias.

## ✨ Funcionalidades

* Busca de notícias por palavra-chave
* Categorias rápidas (Tecnologia, Mundo, Esportes e Negócios)
* Interface minimalista e responsiva
* Cards com imagem, título, descrição e link da notícia
* Atualização dinâmica utilizando API

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* API de Notícias (NewsAPI ou GNews)

## 📁 Estrutura do Projeto

```text
news-site/
│
├── index.html
├── style.css
└── script.js
```
